Welcome to Vulcan

This is my Vulcan theme. Mainly inspired by the AK-47 | Vulcan CS:GO weapon skin. The main idea I had for this static site was making it a lot like a video game interface. The architecture and layout will look like the recent Panorama UI in CS:GO. What makes Panorama very special is the drastic performance improvements over the older UI. The older one was Scaleform and used Flash Player as its engine which is obviously super outdated. Panorama uses HTML/JavaScript which are dynamic, responsive and lightweight.
